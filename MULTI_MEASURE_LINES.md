# 多条测距线功能实现

## 功能概述

实现了多条测距线功能，允许用户同时绘制和管理多条测距线，最多支持20条。

## 核心特性

### 1. 绘制多条测距线
- **中键点击**：开始新的测距线（第一个点）
- **再次中键点击**：完成当前测距线（第二个点）
- **自动递进**：完成一条线后，下一次中键点击自动开始新线
- **最大限制**：最多支持 20 条已完成的测距线

### 2. 删除测距线
- **左键双击测距线**：删除被点击的测距线
- **左键双击信息框**：删除该线对应的信息框
- **右键双击**：取消正在绘制的当前线

### 3. 信息框显示
- **位置**：测距线的中点处显示
- **内容**：距离（km）和方位角（度）
- **背景**：透明背景（不显示黑色背景框）
- **可交互**：可通过左键双击删除

## 代码实现

### 数据结构

```python
# 正在绘制的测距线（最多2个点）
self._measure_points: List[Tuple[RealPoint, Optional[RadarTrack]]] = []

# 已完成的测距线列表（最多20条）
self._completed_measure_lines: List[Tuple[...]] = []

# 测距过程中的临时终点
self._measure_temp_end: Optional[...] = None

# 存储信息框位置（用于双击检测）
self._measure_info_boxes: List[Tuple[float, float, float, float]] = []
```

### 关键函数

#### `_handle_measure_click(mx, my)`
处理中键测距点击：
- 第一次中键：创建新测距线，记录起点
- 第二次中键：完成当前线，保存到 `_completed_measure_lines`

#### `_distance_to_line_segment(px, py, x1, y1, x2, y2)`
计算点到线段的距离，用于检测点击是否在测距线附近。

#### `_draw_measure(painter)`
绘制所有测距线：
- 遍历已完成的线：调用 `_draw_single_measure_line`
- 绘制正在绘制的临时线
- 记录所有信息框位置

#### `_draw_single_measure_line(painter, pt1, pt2, fm, line_idx)`
绘制单条已完成的测距线：
- 绘制线段和端点标记
- 计算距离和方位角
- 在中点显示信息框
- 记录信息框位置用于双击检测

#### `mouseDoubleClickEvent(event)`
处理左键双击删除：
- 检测是否点击在信息框上（精确）
- 检测是否点击在测距线附近（5像素容差）
- 删除对应的测距线

## 使用流程

### 绘制测距线
1. 中键点击屏幕上的第一个位置（地点/航迹）
2. 中键点击第二个位置（地点/航迹）
3. 测距线完成显示，包括距离和方位角信息

### 绘制另一条测距线
1. 中键点击新的起点位置
   - 前一条线保持显示
   - 新线开始绘制
2. 中键点击新的终点位置
3. 新线完成

### 删除测距线
- **方式1**：左键双击测距线本身
- **方式2**：左键双击距离/方位信息框
- **方式3**：右键双击取消正在绘制的当前线（未完成）

## 技术细节

### 信息框背景透明处理
```python
# 透明背景（不画背景，直接画文字）
painter.drawText(int(mid_x - text_w / 2), int(mid_y - text_h / 2), 
                text_w, text_h, Qt.AlignmentFlag.AlignCenter, info_text)
```

### 点到线段距离计算（用于双击检测）
使用向量投影方法：
- 计算点在线段方向上的投影参数 t
- 当 0 ≤ t ≤ 1 时，投影点在线段内
- 超出范围时，取最近的端点
- 容差范围：5 像素

### 测距线存储优化
- **已完成线**：`_completed_measure_lines` 存储完成的线对
- **正在绘制线**：`_measure_points` 存储当前线的起点（和可能的终点）
- **临时终点**：`_measure_temp_end` 在鼠标移动时实时更新

## 性能考虑

|------|------|
